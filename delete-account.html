앱에서 직접 삭제 (권장)

자세AI 앱 내에서 직접 계정을 삭제할 수 있습니다.

  1. 자세AI 앱 실행
  2. 하단 탭 → 프로필 탭 선택
  3. 설정(⚙️) 아이콘 클릭
  4. 하단 계정 삭제 버튼 클릭
  5. 확인 후 영구 삭제 완료

⚠️ 계정 삭제 시 모든 분석 기록, 루틴, 구독 정보가 영구적으로 삭제되며 복구할 수 없습니다.

삭제되는 데이터

  • 회원 정보 (이메일, 이름 등) → 즉시 삭제
  • 운동 분석 기록 → 즉시 삭제
  • 운동 루틴 → 즉시 삭제
  • 업로드한 운동 영상 → 즉시 삭제
  • 결제 및 구독 이력 → 전자상거래법에 따라 5년 보관 후 삭제
